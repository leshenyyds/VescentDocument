# Vescent Document

现代化的文档解决方案，基于 VitePress 构建。

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建生产版本
pnpm docs:build
```

## 项目结构

```
VescentDocument/
├── docs/
│   ├── .vitepress/
│   │   └── config.js    # VitePress 配置
│   ├── public/          # 静态资源
│   │   └── logo.svg
│   ├── guide/           # 指南文档
│   ├── api/             # API 文档
│   ├── changelog.md
│   └── index.md         # 首页
├── package.json
└── README.md
```

## 功能特性

- ⚡ Vite 驱动 - 极速开发体验
- 📝 Markdown 支持 - 简洁的内容编写
- 🎨 主题定制 - 灵活的主题系统
- 🚀 静态生成 - 轻松部署
