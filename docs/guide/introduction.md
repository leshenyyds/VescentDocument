# 介绍

欢迎使用 Vescent Document！

Vescent Document 是一个现代化的文档站点解决方案，基于 VitePress 构建。它提供了快速、简洁、强大的文档编写和发布体验。

## 核心特性

- **Vite 驱动** - 利用 Vite 的极速 HMR，提升开发效率
- **Markdown 扩展** - 在 Markdown 中使用 Vue 组件
- **主题定制** - 灵活的主题配置和自定义能力
- **静态生成** - 预渲染为纯 HTML，SEO 友好

## 为什么选择 Vescent？

1. **开箱即用** - 无需复杂配置，快速启动文档站点
2. **性能优异** - 基于 Vite 构建，页面加载速度快
3. **易于维护** - 使用 Markdown 编写，降低维护成本
4. **扩展性强** - 支持 Vue 组件，可自由扩展功能
